== Introduction ==

Wiking is a web application development platform.  The most interesting part
for end users is the content management system built on top of this platform.

The following sections describe wiking from the perspective of the [user Wiking
website visitor], [cms Wiking CMS user/administrator], [admin web server
administrator] and [devel software developer].
